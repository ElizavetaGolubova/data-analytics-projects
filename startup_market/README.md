# Анализ венчурного рынка: финансирование стартапов

**Задача:** Проанализировать исторические данные о финансировании стартапов, выявить наиболее перспективные сегменты и типы инвестиций.

**Инструменты:** Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy)

**Данные:** 
- `cb_investments.csv` — 54 294 компании, 40 полей (объёмы финансирования по типам, раунды, даты)
- `cb_returns.csv` — возвраты инвестиций по годам (2000-2014)

---

## Что было сделано

### 1. Предобработка
- Нормализация названий столбцов
- Преобразование `funding_total_usd` из текста с запятыми в число
- Заполнение пропусков: текстовые → 'unknown', `mid_funding_at` → средняя дата между first и last
- Удаление дубликатов и строк без финансирования

### 2. Инжиниринг признаков
- **Группы по срокам финансирования:** единичное / до года / более года
- **Классификация сегментов рынка:** массовые (>120 компаний), средние (35-120), нишевые (<35)

### 3. Работа с выбросами
- Определение типичного диапазона финансирования (0.3M - 10.0M USD)
- Выявление выбросов методом IQR **отдельно по каждому сегменту**
- Топ сегментов с аномальным финансированием: Real Estate (17.9%), Technology (16.2%)

### 4. Анализ типов финансирования
- **По объёму:** Venture — лидер (66% всех средств)
- **По популярности:** Seed — самый массовый (используют 60% компаний)
- **По возвратам:** Venture и Private Equity генерируют 85% всех возвратов

### 5. Динамический анализ
- Пик среднего размера раунда — 2005 год (4.5M USD)
- 2014 год: коррекция после перегрева (снижение раундов на 45%)
- **Растущие массовые сегменты:** Medical (+126% в 2014), Apps (+122%), Startups (+118%)

### 6. Анализ возвратов
- Расчёт нормированной доли возвращённых средств (возвраты / финансирование)
- Положительный тренд: Venture и Angel-инвестиции

---

## Ключевые выводы

| Вывод | Данные |
|-------|--------|
| **Venture — основной инструмент** | 66% всех средств, 85% возвратов |
| **Medical — самый быстрорастущий сегмент** | +126% в 2014 году |
| **Рынок перешёл от качества к количеству** | Размер раунда упал с 4.5M → 0.6M USD при росте числа сделок |
| **Angel и Venture — единственные с положительным трендом возвратов** | Остальные типы финансирования снижают эффективность |

---

## Рекомендации (на 2015 год)

1. **Инвестировать в сегмент Medical** — максимальный рост, устойчивый тренд
2. **Использовать Venture-финансирование** — лидер по объёму, возвратам и устойчивости
3. **Дополнительно — Angel-инвестиции** для диверсификации на ранних стадиях
