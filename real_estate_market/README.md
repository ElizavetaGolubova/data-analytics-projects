# Анализ рынка недвижимости СПб и Ленинградской области

**Задача:** Помочь агентству недвижимости из Петрозаводска найти самые перспективные сегменты для выхода на новый рынок.

**Данные:** Архив объявлений Яндекс.Недвижимость за 2014-2019 гг. (схема real_estate).

**Инструменты:** SQL (PostgreSQL), DataLens.

**Основные этапы:**
1. Предобработка данных в SQL: фильтрация выбросов через 99-й перцентиль, обработка пропусков.
2. Расчет ключевых метрик: время активности объявлений, стоимость за м², сезонность.
3. Сравнительный анализ двух регионов: СПб (город-миллионник) vs ЛО (область).
4. Визуализация выводов в дашборде DataLens.

**Ключевые выводы:**
- В СПб быстрее всего продаются 2-комнатные квартиры (50-55 м²) с ценой ~109 тыс. руб./м².
- В ЛО ликвидны 1-комнатные квартиры до 48 м² с ценой ~74 тыс. руб./м².
- Пик активности рынка — октябрь-ноябрь, летом спрос снижается.
- Самый быстрый населенный пункт по продажам — Кингисепп (45 дней), самый медленный — Красное Село (135 дней).

**Файлы в папке:**
- `real_estate_queries.sql` — все SQL-запросы с фильтрацией и категоризацией.
- `real_estate_analysis.pdf` — аналитическая записка с выводами.
- `datalens_dashboard_link.txt` — ссылка на дашборд.
