# Дашборд по выступлениям TED

**Задача:** Помочь организаторам конференций TED проанализировать опыт прошлых мероприятий: популярные темы, реакцию зрителей, длительность выступлений, самых интересных спикеров.

**Инструмент:** DataLens

**Источник данных:** База данных `data-analyst-ted` (таблицы events, speakers, talks)

**Дашборд отвечает на вопросы:**
- Какие темы популярны? Как зрители реагируют (смех, аплодисменты)?
- Какое среднее количество выступающих на конференции?
- Какая средняя длина выступления?
- Кто из выступающих интересен зрителям?

**Функционал дашборда:**
- Фильтры: по стране, конференции, тегу, дате записи
- Топ-10 самых смешных выступлений
- Топ-20 популярных тегов
- Детальная информация о каждом выступлении
